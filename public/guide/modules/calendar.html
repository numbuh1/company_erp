9 Calendar

A shared calendar showing your events alongside approved leaves and overtime requests, with Month, Week, and Day views.

Views

ViewDescription
MonthFull month grid — each day shows event pills. Use Prev/Next arrows to navigate months.
WeekSeven-column grid with full event names, one row per day.
DaySingle day detail showing all events with times.

What Is Shown

ItemColourVisibility
Internal MeetingIndigoEvents you created or are attending
InterviewBlueEvents you created or are attending
Company EventPurpleEvents you created or are attending
Approved LeaveYellowScoped by your leave view permission
Approved OTOrangeScoped by your OT view permission

Filters

Use the filter bar at the top of the calendar to narrow what is shown:

  • Event Types — multi-select tag dropdown. Choose from Internal Meeting, Interview, Company Event, Leave, OT. Leave empty to show all.
  • Location — multi-select tag dropdown. Filter events by one or more locations.

Filters are applied instantly. The URL updates so you can share or bookmark a filtered view.

Creating an Event

🔑
Requires the module calendar permission.
  1. Click New Event on the calendar page.
  2. Fill in the event details (see fields below).
  3. Add Attendants — search by name. Each attendant receives a bell notification.
  4. Optionally attach a File.
  5. Click Save Event.

Event Fields

FieldDescription
NameEvent title
TypeInternal Meeting, Interview, or Company Event
LocationFree-text; saved to the location list for future filtering
Start / EndDate and time — end must be on or after start
DescriptionOptional notes or agenda
FileOptional attachment (max 20 MB)
AttendantsUsers to invite — they are notified by bell notification

Editing an Event

Click any event pill on the calendar to open the edit modal. Change any field and click Save Event.

🔑
You can edit events you created. Users with edit events permission can edit any event.

Deleting an Event

Open the event modal and click Delete Event at the bottom-left of the dialog. A confirmation prompt appears. All attendants receive a cancellation notification.

Booking an Interview

From the Recruitment position page, click the Book Interview button next to an applicant. The event modal opens pre-filled with the position name, applicant name, interview type, and assigned users as attendants. The event is linked to the applicant so it appears on their profile.

Permissions
